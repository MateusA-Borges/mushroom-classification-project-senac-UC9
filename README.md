Mushroom Classification Project developed as part of the UC9 (Curricular Unit 9) coursework at SENAC, focused on applying Machine Learning techniques to classify mushrooms based on their characteristics.

##  Project Description

This project aims to build a Machine Learning model capable of classifying mushrooms into different categories based on their features.  
The model learns patterns from the dataset to distinguish classes and support decision-making through predictive analysis.

The project demonstrates the complete Machine Learning pipeline:
- Data preprocessing 
- Exploratory Data Analysis (EDA)
- Feature analysis and transformation
- Model training and validation
